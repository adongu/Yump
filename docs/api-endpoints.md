# API Endpoints

## HTML API
* `Get /` - loads React web app


## JSON API

### Users
* `POST /api/users` - Creates users
* `PATCH /api/users` - Updates Users

### Session
* `POST /api/session`
* `DESTROY /api/session`


### Businesses
* `GET /api/business/:id`
* `POST /api/business/new`
* `PATCH /api/business/:id`
* `DESTROY api/business/id`

### Pictures
* `POST /api/pictures/new`
* `DESTROY api/pictures/id`

### Reviews
* `POST /api/business/new`
* `PATCH /api/business/:id`
* `DESTROY api/business/id`
